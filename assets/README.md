# Demo asset: `signal-demo.gif`

The main [README](../README.md) references an optional **`signal-demo.gif`** in this folder for the hero section. The file is **not** committed until you record it locally.

## Suggested recording (Windows)

1. Terminal with a readable font size (e.g. Windows Terminal, 14–16 pt).
2. **Install (short clip):** `npx skills add mattbaconz/signal -y` (or show `--list` after add).
3. **Optional:** run `powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\benchmark.ps1` from a clone of this repo.
4. **Optional:** `signal-commit\scripts\commit.ps1 --dry` in a tiny temp git repo.
5. Export as **GIF** (≤ ~15–30 s, reasonable resolution) so README loads quickly.

Save the output as **`assets/signal-demo.gif`** at the repo root path above, then commit if you want it in the README hero.
