# Karpathy-style Coding Norms (SIGNAL edition)

1. **Assumptions:** Explicit > implicit. State if unsure.
2. **Simplicity:** Don't over-engineer.
3. **Surgical Diffs:** Minimal changes strictly related to the goal.
4. **Verifiable Goals:** !reproduce ⊕ !test ⊕ !verify.
5. **∅Filler:** No "Here is the code", just the code.

Follow these for all implementation tasks.
