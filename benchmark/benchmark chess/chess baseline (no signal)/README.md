# Chess baseline (no SIGNAL project context)

This folder is intentionally **empty of `GEMINI.md`**.

Use it with the shared [`../prompt.txt`](../prompt.txt) so Gemini CLI loads **default** project instructions only. The SIGNAL skills may still be **installed** globally (`npx skills add mattbaconz/signal`); without a project `GEMINI.md` that asks for SIGNAL-style replies, behavior stays closer to a normal verbose assistant.

Compare with [`../chess signal (signal skill used)/`](../chess%20signal%20(signal%20skill%20used)/), which adds a project `GEMINI.md` with SIGNAL session defaults.
